# 🕒 Sistem Absensi Digital Perangkat Desa (E-Presensi)

**E-Presensi Desa** adalah solusi modern berbasis web untuk mengelola kedisiplinan dan kehadiran perangkat desa secara akurat, transparan, dan efisien. Lupakan rekapan manual yang membuang waktu; beralihlah ke sistem digital terintegrasi.

## 🚀 Fitur Unggulan

* **✅ Presensi Cepat:** Pencatatan jam masuk dan pulang yang akurat dengan sistem keamanan data.
* **📅 Manajemen Izin & Cuti:** Pengajuan izin (sakit, tugas luar, cuti) langsung melalui sistem dilengkapi unggahan bukti surat.
* **📊 Dashboard Monitoring:** Statistik kehadiran harian dalam bentuk grafik untuk memudahkan kontrol pimpinan.
* **🖨️ Laporan Otomatis (PDF):** Cetak rekapitulasi kehadiran bulanan per perangkat desa atau keseluruhan hanya dengan satu klik.
* **👤 Manajemen User:** Pengaturan data jabatan dan profil perangkat desa yang fleksibel.

---

## 🛠️ Teknologi (Tech Stack)

Sistem ini dibangun dengan teknologi standar industri untuk menjamin keamanan dan performa:
* **Backend:** Laravel 10/11
* **Database:** MySQL
* **Frontend:** Tailwind CSS / Bootstrap
* **Reporting:** DomPDF (Export PDF Standard)
* **Charts:** Chart.js (Visualisasi Data Grafik)
